# 🗄️ Windows Forms Projects — Автоматизация бизнеса

В этой директории собраны десктопные проекты на **C#** и **Windows Forms**, разработанные для автоматизации локальных предприятий, медицинских и образовательных учреждений.

Основной упор в проектах сделан на работу с реляционными базами данных (**Microsoft SQL Server**), реализацию CRUD-операций и применение **Entity Framework Core**.

---

### 🏢 Ключевые проекты (Прикладные решения)

* 🏥 **[Больница](./Больница)** — Информационная система для медицинского учреждения: ведение базы пациентов и учет обращений.
* 🏭 **[ГАЗРЕГИОН](./ГАЗРЕГИОН)** — Система учета для регионального газового предприятия.
* 🎨 **[Детская школа искусств (Балашов)](./Детская_школа_искусств_Балашов)** — Автоматизация внутренних процессов образовательного учреждения.
* 🛒 **[Магнит](./Магнит)** — Учебная система управления складом и продажами для розничной сети.
* 🚜 **[КФХ](./кфх)** — Учет ресурсов, техники и продукции крестьянского фермерского хозяйства.

---

### 📚 Учебные проекты и база знаний

Остальные папки в этой директории (каталоги с номерами, `Новая папка`, `Программа`, `бд` и т.д.) содержат:
* Лабораторные работы по изучению компонентов Windows Forms.
* Тестовые проекты для отработки алгоритмов и структур данных.
* Примеры подключения к локальным базам данных и написания SQL-запросов.

> **Технологический стек директории:** C#, .NET Framework, Windows Forms Designer, Entity Framework Core, ADO.NET, MS SQL Server.
